# InventoryPro Readability Upgrade

This version increases the practical font size across the complete app while preserving the existing modern layout.

Updated areas:
- Sidebar navigation, profile and alerts
- Topbar, search and notification panel
- Page headings, buttons and filters
- Dashboard KPI cards, charts and list widgets
- All tables, chips, status badges and pagination
- Products, categories, orders, inventory, suppliers and customers
- Reports, activity, settings, modals and forms
- Responsive card columns for medium-size laptop screens

The update is placed at the bottom of `src/App.css`, under the heading `READABILITY UPGRADE`.
